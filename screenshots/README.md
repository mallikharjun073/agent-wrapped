# Screenshots

Drop your generated PNGs in this folder so the main README's images render on GitHub.

## How to generate

Run the tool with `--html`, then screenshot the browser page:

```bash
agent-wrapped --html --days 30
```

Suggested files (referenced from the top-level README):

- `report-hero.png` &mdash; hero + card grid (top of the page)
- `report-terminal.png` &mdash; a terminal screenshot showing the ASCII version
- `report-insights.png` &mdash; the "Highlights & growth edge" section only

## Tips

- Set your terminal to a **dark theme** for the terminal shot &mdash; matches the HTML theme.
- Crop each PNG to ~1600&times;900 or 1280&times;720 for good preview aspect ratio on GitHub.
- Zoom the browser to **90%** before capturing the HTML page &mdash; more fits on screen without losing legibility.
- Use a **1080&times;1350** portrait crop for social sharing (X, LinkedIn).

Once you drop the PNGs here, the images in the main README will render automatically on your repo home page.
